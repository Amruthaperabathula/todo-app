# Focus List — Full-Stack To-Do Application

A full-stack task management web application built using React, TypeScript, Node.js, Express, and PostgreSQL. The application helps users manage daily tasks with due dates, task filtering, and productivity tracking features.

---

## Features

- Create, edit, and delete tasks
- Mark tasks as completed
- Add due dates and priority levels
- Filter tasks by status and priority
- View live productivity statistics
- Responsive and user-friendly interface
- Persistent task storage using PostgreSQL

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React, TypeScript, Vite |
| Styling | Tailwind CSS |
| Backend | Node.js, Express.js |
| Database | PostgreSQL |
| ORM | Drizzle ORM |

## Installation & Setup

```bash

# Install dependencies
npm install

# Run project
npm run dev
```
